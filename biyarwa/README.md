# About Biyaliquid

Biya 是一条专注于去中心化交易的高性能专用链，致力于构建一个低延迟、可扩展、链上撮合的去中心化交易基础设施。\
应用场景

## Technical overview

* 去中心化撮合引擎
* 低延迟结算
* 高 TPS
* 原生支持 EVM 合约与 Wasm 合约（WasmX 扩展）
